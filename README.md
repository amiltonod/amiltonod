## 🏆 Projetos em Destaque

### 1. **ColetaFlow** 🚛
_Sistema de programação inteligente de coletas de resíduos_

**O Problema:**
- Planejamento semanal feito manualmente em Excel (30 min/semana)
- Sem histórico, sem validações, propenso a erros
- Perda de ~30h/ano só em cliques

**A Solução:**
- Geração automática em **5 segundos** vs 30 minutos manual
- Sistema web com drag & drop, histórico completo, auditoria
- Ganho: **2.5h/mês** de tempo operacional

**Tech Stack:**
- Backend: **FastAPI** + SQLAlchemy + SQLite
- Frontend: Jinja2 + HTML5 + JavaScript (drag & drop)
- Documentação: 5 guias técnicos (ARCHITECTURE, API, CODE_REVIEW, ROADMAP, SETUP)

**Destaques:**
- ✅ Lógica defensiva (triple-layer anti-duplicidade)
- ✅ Importação Excel com validação robusta
- ✅ Fechamento automático com auditoria
- ✅ Separação clara de responsabilidades (Routes / Services / Models)
- ✅ Code review + roadmap de melhorias documentados

📂 [Ver Repositório](https://github.com/amiltonod/coleta-flow) | 📖 [Ver Documentação](https://github.com/amiltonod/coleta-flow/tree/refatoracao/docs)
